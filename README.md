# Drift Python SDK

Python SDK for Tusk Drift instrumentation and replay.

## Installation

```bash
pip install tusk-drift-python-sdk
```

### With Framework Support

```bash
# Flask support
pip install tusk-drift-python-sdk[flask]

# FastAPI support
pip install tusk-drift-python-sdk[fastapi]
```

## Requirements

- Python 3.12+
